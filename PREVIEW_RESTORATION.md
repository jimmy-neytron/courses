# Восстановление предпросмотра уроков

Предпросмотр снова использует секционную структуру старого учебного плеера:

- теория;
- аудирование;
- практика;
- проверка знаний.

Блоки оформлены отдельными карточками и интерактивными заданиями. Для `audio` и
`listening` используется аудиофайл, когда он указан. Когда URL отсутствует,
текст блока озвучивается через Web Speech API браузера с английской локалью
`en-US` по умолчанию и скоростью `0.88`, как в исходной реализации.

Текст для озвучки хранится в `publicContent.text`. URL записи необязателен.
Черновики по-прежнему исключаются из обычного предпросмотра и доступны только в
авторском режиме `?mode=author`.
