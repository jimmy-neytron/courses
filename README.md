# Курсор — Course Platform

Vue 3 + Vite приложение по ТЗ из `course-platform-codex-kit.zip`. Код организован в Nuxt-подобной структуре: `pages`, `layouts`, `components`, `composables`, `stores`, `services`, `types`, `utils`, `assets`.

## Запуск

```bash
pnpm install
pnpm dev
```

Без `.env` интерфейс работает на демонстрационных данных. Для Supabase скопируйте `.env.example` в `.env` и заполните publishable-параметры. SQL-схема находится в `supabase/sql`.
